## 🟡 전체 흐름 구축 (농가 → 상품 → 주문 → 배송)

```
① 농가 등록
   농가명 · 대표자명 · 연락처 · 주소 · 택배사 선택 · 인증정보

② 상품 등록
   상품 등록 시 "파트너 농가" 선택 → 농가 자동 연결

③ 주문 처리
   농가별 필터 → 발주 엑셀 다운로드 (해당 농가 택배사 양식 자동 변환)

④ 배송 처리
   농가가 송장번호 전달 → 주문관리에서 입력 → 소비자 배송추적 자동 연결
```
