📌 레퍼런스 해석 (틀리면 정정)

온브릭스 픽카드가로 스크롤·정사각 이미지·썸네일 우하단 장바구니 아이콘·상품명→가격·새벽배송 태그 → 메인 델리오픽, 찜 상품카드

온브릭스 스토리제목+더보기 / 칩 탭 / 카드(좌상단 날짜뱃지, 카테고리 라벨+제목) → 델리오 라운지
온브릭스 중간배너회색 배경, 좌측 텍스트 + 우측 상품 이미지 → 메인 중간배너
온브릭스 상품상세 하단바[♡][🎁선물하기][구매하기(검정/연두글씨)] → 상품상세 하단 버튼
온브릭스 장바구니체크박스가 썸네일이 아닌 "배송그룹" 줄 좌측에, 연두 체크, 큼 → 장바구니 체크박스
온브릭스 1:1문의 / 포인트내역카운트+문의하기 / 사용가능 크게·기간탭·내역 → 마이페이지
어글리어스 담기이미지 하단 "🛒 담기" 테두리 버튼 + 설명→상품명→할인%·원가→가격→별점 → 퀵가이드 가격하단·담기
어글리어스 블로그이미지+하단 흰영역 태그/제목, 가로 스크롤 → 브랜드 직송관 카드
쿠캣 마이페이지프로필+등급, 후기/포인트/쿠폰 카운트, 메뉴 리스트(구분선 꽉참) → 마이페이지 전체 톤
프레딧 쿠폰쿠폰등록 입력 / 쿠폰카드(할인율 크게, n일후소멸 빨강) → 마이페이지 쿠폰내역
에이블리 / 오늘의집주문·예약내역 카운트 / 주문카드+버튼 → 배송조회 상단 / 하단
포스텔러그라데 헤더+레벨+진행바 → 취향진단 화면
아르르 주문결제·배송지주문상품→주문자→배송지→요청사항, 고정 결제바 → 주문/결제·회원정보수정·배송지관리

1. 메인

  • 로고 크기 살짝 축소
  • 상단 카테고리 분류: 색상 블랙 + 두께 얇게
  • 메인배너 좌우 스크롤 가능
  • 델리오픽: 상품명↔설명 간격 동일(할인가 없어도 동일 높이)
  • 델리오픽: PC처럼 + 가로 스크롤로 일부만 보이게 (ref: 온브릭스 픽카드)
  • 델리오픽: 썸네일 우하단 장바구니 아이콘 추가 (ref: 온브릭스 픽카드 / 어글리어스 담기)
  • 델리오픽: 썸네일 좌상단 산지직송/자사배송 태그 + 아이콘 PC와 동일
  • 퀵가이드: 태그 = pill 칩 (ref: 온브릭스 PICK 태그)
  • 퀵가이드: 가격 하단 "담기" 버튼(두께 비슷) (ref: 어글리어스 담기)
  • 퀵가이드: 상품 이미지 정사각형 통일
  • 브랜드 직송관: 카드 = 이미지+하단 태그/농가·상품명 (ref: 어글리어스 블로그)
  • 브랜드 직송관: 배경 회색 → 흰색
  • 중간배너: 좌텍스트+우이미지 (ref: 온브릭스 중간배너)
  • 델리오 라운지: 레이아웃 (ref: 온브릭스 스토리)
  • 각 파트 소제목/대제목/상품 간격 일정(리뷰 하이라이트 넓음 체크)
  • 각 파트 간 간격 줄이기(허전함) (ref: 온브릭스)

2. 푸터

  • 취향부분 회색 경계선 아래 실선 삭제 + 사업자 부분 간격 좁히기
  • 고객센터 번호 크기 줄이기

3. 검색

  • 인기/최근 검색어 태그 테두리 얇게
  • 태그 테두리 색상·두께 PC와 동일하게

4. 장바구니

  • 수량 +/- 색상 검정
  • 수량박스·가격·X 간격 일정(숫자와 체크박스 너무 가까움)
  • 상품명·옵션·수량박스 간격 일정
  • 체크박스 위치: 썸네일 좌측 → 배송그룹 줄 좌측 (ref: 온브릭스 장바구니)
  • 체크박스 모양 온브릭스처럼 + 살짝 키우기 (ref: 온브릭스 장바구니)
  • 주문요약: 상품합계↔배송비 사이 실선 삭제
  • 배송비 색상 검정
  • 결제예정금액: PC처럼 쿠폰 적용금액 추가(빨강)

5. 주문/결제

  • 레이아웃 (ref: 아르르 — 주문상품→주문자→배송지→요청사항→고정 결제바)
  • 할인혜택의 "배송비 쿠폰" 항목 삭제 (배송비 무료라 불필요)

6. 카테고리

  • 상단 배너 배경 화이트
  • 퀵버튼(on/off) PC와 동일 + 태그 디자인 (ref: 쿠켓)
  • 대분류 선택 시 글자·화살표 색상 검정
  • 대분류 클릭 시 배경색 제거, 하단 소분류만 회색배경, 소분류 글씨 검정
  • 소분류 구분선 삭제 + 글자 크기 살짝 축소
  • 대분류 서비스 하단 여백 추가
  • 아무거나 클릭 후 뒤로가기 시 카테고리로 복귀 동작

7. 배송조회 (= 마이페이지 주문배송내역과 동일 화면 확인 필요)

  • 상단: 배송중/배송완료/취소·환불 카운트, "취소/반품"→"취소/환불", 숫자 파랑→검정 (ref: 에이블리)
  • 하단 주문카드: 리뷰쓰기(아이콘삭제·테두리/글씨 빨강)/문의/재구매, 장바구니 삭제, 태그 가로 확장 정렬 (ref: 오늘의집)
  • 결제완료 상태: 문의/주문취소 버튼, 배송완료 위 배송조회 노출

8. 찜

  • 상단 마이페이지/뒤로가기 삭제, "나의 위시리스트" 위로
  • "나의 위시리스트" 하단 실선 삭제
  • 썸네일 정사각형
  • 상품카드 = 델리오픽처럼(가격아래 태그/리뷰, 산지직송·자사배송 좌상단, 가격하단 NEW 태그) (ref: 온브릭스 픽카드)

9. 마이페이지

  • 상단 우측 장바구니 삭제
  • 전체 느낌 (ref: 쿠캣), 적립금→포인트
  • 회색 구분선 끝까지 꽉 차게(소분류 실선 삭제)
  • 파트 구분 소제목: 글씨 얇게 + 크기 키우기
  • "적립금 내역" → "포인트 내역"
  • 각 파트 상단 뒤로가기 화살표 + 장바구니 삭제
  • 포인트 내역 (ref: 온브릭스 — 사용가능 크게/기간탭/내역)
  • 쿠폰내역 (ref: 프레딧) + "쿠폰 선물하기" 삭제
  • CS/환불내역 삭제 → 주문배송내역에서 결제후 문의/주문취소, 배송완료 위 배송조회
  • 1:1문의 (ref: 온브릭스 — 카운트+문의하기)
  • 취향진단 "나를 위한 추천상품" PC처럼 off 가능
  • 취향진단 이미지 저장 시 MBTI처럼 한 이미지에
  • 취향진단 화면 (ref: 포스텔러)
  • 맞춤 상품 현재는 안 보이도록
  • 최근본상품 이미지 동일하게
  • 내가쓴리뷰: 아래는 삭제(상단에 표기되므로)
  • 친구초대 쿠폰: 가로로 안 늘어지게
  • 누적 쿠폰 금액 빨강
  • 코드 복사 시 "코드가 복사되었습니다" 표기
  • 회원정보 수정: 레이아웃 (ref: 아르르), 마케팅 동의 체크만 (ref: 온브릭스 [필수]/[선택] 연두체크)
  • 배송지 관리 (ref: 아르르 배송지 목록/추가 모달)

10. 하단바 / 네비게이션

  • 하단바 홈 아이콘 교체 (ref: 온브릭스 홈 아이콘)
  • 다른 탭 눌러도 홈 버튼에 색 계속 남음 → 안 남게 동작
  • 카테고리 클릭 후 홈/다른 하단탭 눌러도 안 보이는 문제 동작
  • breadcrumb(홈>사과/배>…) 따라오는 기능 삭제 동작

11. 상품 상세

  • 선물 아이콘 빨강
  • 가격 아래 배송방법↔배송비 사이 포인트 부분 실선 삭제
  • 가격 "원" 뒤 ~(물결) 삭제
  • 썸네일 좌우 스크롤 가능
  • 서브 사진↔농장 간격 띄우기
  • 할인% 자릿수 달라도 최종가격 정렬 맞추기
  • 포인트↔옵션 사이 회색 구분선 가로 꽉 채우기
  • 옵션 클릭 시 파란 박스 안 나오게
  • "필수 옵션을 선택해주세요" 짤림 → 단락 안 나눠지게
  • 후기 "최신순" 하단 실선 삭제
  • 바로구매 클릭 시 나오는 화면 박스 우측 빈공간 넓히기
  • 후기 상단 우측 "추천순" 버튼 파랑→회색 (ref: 아르르)
  • 찜: 상품상세에서 [♡][🎁선물하기][구매하기] 배치 (ref: 온브릭스 하단바)
📋 확인 필요 2가지
① 배송조회 화면 = 마이페이지 "주문배송내역"과 같은 화면? (같다면 하나로 통합 작업)
② 퀵가이드 태그를 온브릭스 PICK 칩 스타일로 가면 될지? (카테고리 퀵버튼 태그 = 쿠켓)
✅ 레퍼런스 전부 매칭 완료 — 이미지 대기 없이 전 항목(약 70개) 작업 가능.
