## 📌 레퍼런스 해석 (내 이해 — 틀리면 알려줘)
- **온브릭스 픽카드**: 가로 스크롤, 정사각 이미지, 썸네일 우하단 장바구니 아이콘, 상품명→가격, 새벽배송 태그, 하단 댓글수
- **온브릭스 PICK 태그**: 가로 스크롤 pill 칩(선택=연두 채움, 나머지=연회색)
- **온브릭스 스토리**: 제목+더보기 / pill 칩 탭 / 카드(좌상단 날짜뱃지, 이미지, 카테고리 라벨 빨강+제목)
- **온브릭스 중간배너**: 회색 배경, 좌측 텍스트(타이틀+할인문구) + 우측 상품 이미지
- **온브릭스 상품상세 하단바**: [♡] [🎁선물하기] [구매하기(검정/연두글씨)] — 찜·선물·구매 3버튼
- **온브릭스 장바구니**: 체크박스가 썸네일 좌측이 아니라 "배송그룹(일반배송)" 줄 좌측에, 연두 체크, 살짝 큼
- **온브릭스 1:1문의**: 상단 전체/답변완료/답변대기 카운트 + "문의하기" 버튼
- **온브릭스 포인트내역**: (적립금 화면 ref) 사용가능 포인트 크게(빨강), 적립예정/소멸예정, 기간탭, 내역리스트
- **어글리어스 담기**: 상품카드 이미지 하단 "🛒 담기" 테두리 버튼 / 그 아래 설명(작은회색)→상품명→할인%+원가→가격→별점
- **어글리어스 브랜드블로그**: 카드 이미지 위, 하단 흰영역에 카테고리 태그+제목, 가로 스크롤
- **쿠캣 마이페이지**: 프로필+등급, 후기/포인트/쿠폰 카운트 3분할, 메뉴 리스트(구분선 꽉참)
- **프레딧 쿠폰**: 쿠폰등록 입력칸 / 쿠폰카드(할인율 크게, All 우측, n일후소멸 빨강, 사용기간)
- **에이블리 주문·예약내역**: 탭(주문/예약) + 배송중/배송완료/취소반품 카운트
- **오늘의집 주문내역**: 상단 "리뷰 포인트" 배너 / 주문카드 + 버튼(리뷰쓰기/문의/재구매/장바구니)
- **포스텔러**: 그라데 헤더 + 레벨/진행바 + 레벨별 혜택 (취향진단 화면 느낌)
- **아르르 주문/결제**: 주문상품→주문자(이름/연락처/이메일)→배송지(기본배송지 카드+변경)→배송요청사항, 하단 고정 결제바
- **아르르 추천순 버튼**: 파랑 아님, 회색 토글
- **배송지 목록/추가 모달**: 전체n건+정렬, 기본배송지 카드(수정|삭제), +배송지 추가
